# **App Name**: React TaskMaster

## Core Features:

- Task Creation: Allow users to create new tasks with titles, descriptions, due dates, and priority levels.
- Task Listing: Display tasks in a list format, sorted by due date or priority.
- Task Editing: Enable users to edit task details.
- Task Completion: Allow users to mark tasks as complete.
- Task Deletion: Allow users to delete tasks.
- Local Storage Persistence: Save task data to local storage to maintain data between sessions.
- AI Enhanced Task Description: Use generative AI to provide a full, detailed description of the task based on title of the task. This functions as an optional tool within the task description creation workflow.

## Style Guidelines:

- Primary color: Light Lavender (#E6E6FA) for a calm, organized feel.
- Background color: Soft White (#FAFAFA) to provide a clean, distraction-free background.
- Accent color: Pale Green (#D3FFCE) to signal the completion of the task and other successful operations
- Body and headline font: 'Inter' (sans-serif) for clear, readable text.
- Use simple, intuitive icons to represent task actions.
- Employ a clean, single-column layout to minimize distractions and improve focus.
- Subtle animations to acknowledge task completion and other actions, creating a polished user experience.
